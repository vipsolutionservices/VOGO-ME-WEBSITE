Ce conține această pagină

Ai tot fluxul necesar pentru a porni rapid pluginul:

  • link direct de descărcare pentru fișierul ZIP;
  • instrucțiuni clare pentru instalare în WordPress;
  • instrucțiuni detaliate pentru configurarea pluginului.

Instrucțiuni instalare plugin în WordPress

  1. 01

    Autentifică-te în panoul de administrare WordPress (/wp-admin) cu un cont care are drepturi de administrator.

  2. 02

    Accesează meniul Plugins → Add New Plugin.

  3. 03

    Apasă pe butonul Upload Plugin.

  4. 04

    Selectează fișierul vogo-plugin.zip descărcat din această pagină.

  5. 05

    Apasă Install Now și așteaptă finalizarea instalării.

  6. 06

    Apasă Activate Plugin pentru activarea pluginului.

Notă: dacă WordPress afișează erori de permisiuni la instalare, verifică drepturile de scriere pentru directorul /wp-content/plugins.

Instrucțiuni detaliate configurare plugin în WordPress

  1. 01

    Deschide pagina de setări a pluginului din meniul WordPress.

    • După activare, intră în secțiunea dedicată pluginului VOGO din meniul admin.
    • Confirmă că pluginul este activ și vizibil în lista de pluginuri instalate.
  2. 02

    Completează câmpurile de configurare disponibile în pagina pluginului.

    • Introdu valorile necesare exact în câmpurile lor dedicate.
    • Verifică fiecare valoare înainte de salvare pentru a evita erori de integrare.
  3. 03

    Salvează configurația.

    • Apasă butonul Save / Save Changes.
    • Verifică mesajul de confirmare afișat de WordPress după salvare.
  4. 04

    Verifică funcționarea pluginului.

    • Reîncarcă pagina de setări și confirmă că valorile salvate rămân persistente.
    • Testează funcționalitatea principală a pluginului în zona site-ului unde trebuie să fie activă.

Recomandare: după orice actualizare a pluginului, revizuiește setările și rulează un test rapid de funcționalitate.
